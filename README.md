#   End-to-End Retail Product Analysis with SQL | BigBasket Case Study
### Project overview
BigBasket is one of India’s largest online grocery platforms, offering products across multiple categories and brands.
This project analyzes the retail product catalog snapshot to uncover insights related to category dominance, pricing strategy, discount behaviour, and customer ratings using SQL-driven analysis.

#### Objectives:
- Understand category-wise product distribution
- Analyze discounting behavior across categories
- Study the relationship between product price and customer ratings
- High-value and low-value product segments
